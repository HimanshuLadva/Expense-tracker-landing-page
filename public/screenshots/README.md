# Screenshots Directory

This directory contains the application screenshots used in the landing page.

## Required Files:

Save your 8 application screenshots here with these exact filenames:

1. `dashboard.png` - Analytics Dashboard page
2. `accounts.png` - Accounts management page
3. `categories.png` - Categories management page
4. `transactions.png` - Transactions list page
5. `budget.png` - Budget management page
6. `reminders.png` - Reminders page
7. `users.png` - User management page
8. `profile.png` - User profile page

## Image Specifications:

- **Format:** PNG or JPG
- **Recommended size:** 1920x1080 or similar (16:9 or 16:10 aspect ratio)
- **File size:** Keep under 500KB each for optimal performance
- **Quality:** High quality but optimized for web

## How to Add Screenshots:

1. Save each screenshot from your chat/email with the exact filenames above
2. Place them in this directory (`public/screenshots/`)
3. Restart the development server if it's running
4. Visit http://localhost:4201 to see them on the landing page

## Where They're Used:

- **Hero Section:** Shows `dashboard.png` as the main preview
- **Screenshots Gallery:** Shows all 8 screenshots with interactive thumbnails

## Optimization Tips:

For better performance, optimize images using:
- **TinyPNG:** https://tinypng.com
- **Squoosh:** https://squoosh.app
- **ImageMagick:** `magick convert input.png -quality 85 output.png`
